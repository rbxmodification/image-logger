How to use Laced v2.14
========================================================================
1. Open laced_install.exe and set up as normal.
2. Open the settings and add your webhook.
3. Add the image you want to use for the logger.
4. Build your Uni.jpg file and save it.
5. Send the jpg to someone and wait for them to open it.
6. Enjoy.
